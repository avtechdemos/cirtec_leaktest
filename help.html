Clearlink Leak Tests — Help

Baxter Clearlink parts are checked on Uson Sprint mD machines using a pressure decay test: fill the part with air, let it settle, then watch whether pressure drops during a short Measure window.

Each chart below uses the same 5-step template (ideal path in the background) with the actual cycle drawn on top, so you can see where a test stayed on track or diverged.

Pass, Fail (Gross), and Abort examples are from machine 071709-001N on 07/07/2026. The Fail (Measure ΔP) example is illustrative — these sample logs only contain Gross fails, not high Measure ΔP fails. Chart X-axis is test step order, not exact clock seconds.
