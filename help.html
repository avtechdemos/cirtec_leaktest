Clearlink Leak Tests — Help

Baxter Clearlink parts are checked on Uson Sprint mD machines using a pressure decay test: fill the part with air, let it settle, then watch whether pressure drops during a short Measure window.

Each chart below uses the same timed template (ideal path in the background) with the actual cycle drawn on top, so you can see where a test stayed on track or diverged.

Dashboard statuses follow process order: Abort → Unstable → Fail → Pass, depending on how far the cycle got. Abort, Unstable, and Pass examples below are from machine 071709-001N on 07/07/2026. The Fail (Measure ΔP) example is illustrative. Chart X-axis uses the machine stage times (0.9 s each). Raw logs may still say Fail, Gross for Unstable.

Machine settings (Clearlink Standalone)

These are the acceptance limits and stage times used on the tester. Charts shade the 7.5–8.5 psig band across Fill and Stabilize.

Stage Duration Acceptable If outside / over limit
Fill 0.9 s 7.5–8.5 psig Abort if < 7.5 or > 8.5
Stabilize 0.9 s 7.5–8.5 psig Unstable if < 7.5 or > 8.5
Measure 0.9 s ΔP ≤ 1.0 psig Fail if ΔP > 1.0; Pass if ≤ 1.0
Vent 0.9 s Exhaust to 0 —

Total timed window for Fill → Stabilize → Measure → Vent is 3.6 s. Start at 0 s is schematic on the charts.

Where the test can stop

Status depends on how far the cycle got. The chart below marks the stop point on the same timed template (with the 7.5–8.5 psig Fill/Stabilize band) and the dashboard result that appears.

Stop point Dashboard status What happened
During Fill (0.0–0.9 s) Abort Pressure outside 7.5–8.5 psig (low or high). Common raw: Abort, Fill Low
During Stabilize (0.9–1.8 s) Unstable Pressure left 7.5–8.5 psig while settling (low or high). Raw often: Fail, Gross
End of Measure — ΔP > 1.0 psig Fail Measure completed; pressure drop exceeded 1.0 psig (raw: Fail, Max limit)
End of Measure — ΔP ≤ 1.0 psig Pass Measure completed; drop within the 1.0 psig limit
Help guide for Clearlink / Baxter leak-test logs. Dashboard statuses: Abort → Unstable → Fail → Pass by process step. Fill/Stabilize band 7.5–8.5 psig; Measure Pass if ΔP ≤ 1.0 psig. Each stage (Fill, Stabilize, Measure, Vent) is 0.9 s. Template path assumes pressure stays in-band with no Measure ΔP. Start and vent at 0 are schematic. Measure-start pressure is calculated as Final Pressure + Measure ΔP when Measure completed.
